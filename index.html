Требования к баннеру

Нужно создать баннер для рекламы на маркетплейсе Ozon. На баннере должны быть изображены шкаф, логотип и фон. Разрешение: 1080 x 400 пикселей (с охранными полями: 926 x 400 пикселей). Максимальный размер файла: до 150 КБ.

Ниже представлены необходимые для баннера ресурсы — два логотипа: для светлых и для тёмных фонов. Несколько вариантов шкафов на выбор.
